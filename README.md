# js-lib-boilerplate
This is a boilerplate for developing JavaScript library.

## Tools
module bundler: __Rollup__  
transpiler: __Babel__  
testing tool: __Jest__  
linter: __ESLint__

## Commands
```
# build
$ npm run build

# build and watch
$ npm run dev

# test
$ npm run test

# test coverage
$ npm run test:cov
```
