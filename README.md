# js-lib-boilerplate
This is a boilerplate for developing JavaScript library.

## Tools
module bundler: __rollup__  
transpiler: __babel__  
testing tool: __jest__  
linter: __eslint__

## Commands
```
# build
$ npm run build

# build and watch
$ npm run dev

# test
$ npm run test

# test coverage
$ npm run test:cov
```
